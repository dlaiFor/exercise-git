# exercise-git


standard command 

1, git clone  [ address ]

2, git log

3, git add [ file name ]  or .   [ . is add all change]

4, git commit 

5, git push origin master

6, git pull origin master

git 和 svn 最大的区别就是一个是分布式，一个是中心式管理，根本就是 git的没有中心的概念，全部都是仓库，也就是你的历史记录，变动
每个仓库都有，中心仓库目前主要应用在一些大的游戏上面，因为媒体资源很大，由服务器统一管理

git branch -a 查看当前分支  -a为包括远程分支

git checkout [ branch ]

git merge [ branch ]  合并 [ branch ] 到当前分支
